Project A:智能识别移动小车
Advantages:
1. 所有零件已打包只需组装
2. 车上有预留的位置安装外设
3. 给出了阶段性的参考教程
4. 有客服提供帮助

Disadvantages：
1. 流程不够详细，代码难懂
2. 无相关的活跃论坛
3. 遇到问题很难解决
4. 每个阶段具体的操作不够明确

Project B:DonkeyCar(驴车)
Advantages:
1. 内容从选材到成品，具体操作详细
2. 项目知名度高
3. 外网有对应的项目论坛
4. 全流程有其他组走通

Disadvantages：
1. 小车费用贵，如果DIY需要自己选配零件，担心耗时长
2. 内容可能过多，时间可能不够
3. 组装过程出现问题不好求助
